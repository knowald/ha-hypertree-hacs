# HA Hypertree for Home Assistant

HACS distribution repository for [HA Hypertree](https://github.com/knowald/ha-hypertree) - interactive tree visualizations for Home Assistant entity hierarchy.

## Installation

1. Open HACS in your Home Assistant instance
2. Add this repository as a custom repository (category: Integration)
3. Search for "HA Hypertree" and install
4. Restart Home Assistant
5. Go to Settings > Devices & Services > Add Integration > search "HA Hypertree"
6. A "Hypertree" entry appears in your sidebar
